# Yii3 社区手册

Yii3 社区手册是一本开源书籍，包含了关于 [Yii3](https://www.yiiframework.com/) PHP 框架的技巧和窍门。

- Yii 社区创建了这本手册。
- Yii 核心团队成员负责策划和编辑。

欢迎提交你自己的文章。团队成员将审查它，提供反馈并以最佳方式合并。

本书遵循 [Yii 文档条款](https://www.yiiframework.com/license#docs)。
